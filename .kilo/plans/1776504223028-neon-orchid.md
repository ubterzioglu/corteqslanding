## Implementation Steps

### Step 1: Prerender (CRITICAL - diger tum SEO buna bagli)

**Dosya:** `vite.config.ts`, `package.json`

1. `vite-plugin-prerender` kur:
   ```
   npm install -D vite-plugin-prerender
   ```
2. `vite.config.ts`'e prerender plugin ekle:
   - Routes: `/` (ana sayfa)
   - StaticDir: `dist`
   - RenderAfterDocumentEvent veya renderAfterTime ile bekle
3. Build test et: `npm run build` sonrasi `dist/index.html` iceriginin dolu oldugunu dogrula

### Step 2: index.html Meta + Canonical + OG Fix

**Dosya:** `index.html`

1. `<link rel="canonical" href="https://corteqs.net/" />` ekle
2. `<meta name="robots" content="index, follow" />` ekle
3. `<meta property="og:url" content="https://corteqs.net/" />` ekle
4. `<meta property="og:site_name" content="CorteQS" />` ekle
5. `<meta property="og:locale" content="tr_TR" />` ekle
6. OG image'i `https://corteqs.net/og-image.png` olarak guncelle
7. `twitter:site` -> `@corteqsx` olarak degistir
8. `twitter:title` ve `og:title` -> "CorteQS - Diaspora Connect Platformu" (keyword-rich)
9. `<title>` -> "CorteQS - Turk Diaspora Network Platformu | Diaspora Connect"
10. `<meta name="description">` -> keyword-rich description guncelle (max 160 char)

### Step 3: robots.txt + sitemap.xml

**Dosya:** `public/robots.txt`, `public/sitemap.xml`

1. robots.txt guncelle:
   ```
   User-agent: *
   Allow: /
   Disallow: /admin
   Sitemap: https://corteqs.net/sitemap.xml
   ```
2. sitemap.xml olustur:
   ```xml
   <?xml version="1.0" encoding="UTF-8"?>
   <urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
     <url>
       <loc>https://corteqs.net/</loc>
       <changefreq>weekly</changefreq>
       <priority>1.0</priority>
     </url>
   </urlset>
   ```

### Step 4: JSON-LD Structured Data

**Dosya:** Yeni `public/schema.jsonld` veya `index.html` icinde inline

Tek bir `@graph` blogu ile:
1. **Organization** schema: name, url, logo, sameAs (LinkedIn, Instagram, X), contactPoint (info@corteqs.net)
2. **WebSite** schema: name, url, potentialAction (SearchAction placeholder)
3. **WebPage** schema: name, description, url, isPartOf
4. **FAQPage** schema: 3-5 soru-cevap (GEO icin kritik):
   - "CorteQS nedir?"
   - "Diaspora network neden onemli?"
   - "CorteQS'e nasil kaydolabilirim?"
   - "Sehir elcisi nedir?"
   - "Onursal kurucu programi nedir?"
5. **Speakable** schema: AI assistant'lar icin (GEO)

Inline `<script type="application/ld+json">` olarak `index.html`'e ekle.

### Step 5: Internal Navigation Links

**Dosya:** `src/components/HeroSection.tsx`

Mevcut sadece `#kategoriler` linki var. Eklenecekler:
- `#hakkinda` -> AboutSection'e scroll (id zaten var)
- `#elciler` -> AmbassadorSection'e scroll (id zaten var)
- `#destek` -> SupportSection'e scroll (id zaten var)

Hero'nun altindaki stats bar yanina veya ayri bir mini nav olarak ekle.

### Step 6: External Authority Links

**Dosya:** `src/components/AboutSection.tsx`

AboutSection'e 2-3 dofollow external link ekle:
- Wikipedia "Turkish diaspora" -> `https://en.wikipedia.org/wiki/Turkish_diaspora`
- OECD veya UN migration report linki

Ornek: "Dunya genelinde 8.8 milyon Turk disasporasina" ifadesini Wikipedia linki ile zenginlestir.

### Step 7: Heading Structure Fix

**Dosya:** `src/components/BloggerSection.tsx`

- `<h4>` taglerini `<h3>` olarak degistir (h2 altinda h4 olmamali)

### Step 8: Image Alt Text Fixes

**Dosyalar:** Multiple components

1. HeroSection.tsx: `"Dunya sehirleri"` -> `"Dunya genelinde diaspora topluluklarini birlestiren CorteQS platformu"`
2. AboutSection.tsx: `"Diaspora toplulugu bulusmasi"` -> `"Turk diasporasi topluluk bulusmasi ve is birligi etkinligi"`
3. AmbassadorSection.tsx: `"Sehir elcisi"` -> `"CorteQS sehir elcilisi - yerel diaspora temsilcisi"`
4. RegisterInterestForm.tsx: bos alt `""` -> `aria-hidden="true"` ekle (decorative)
5. BackerForm.tsx: bos alt `""` -> `aria-hidden="true"` ekle (decorative)

### Step 9: Content Expansion (GEO)

**Dosya:** `src/components/AboutSection.tsx`

AboutSection'e GEO-friendly icerik ekle:
- Definition block: "CorteQS nedir?" -> net tanim paragrafi
- Problem-cozum yapisi: diaspora sorunlari -> CorteQS cozumu
- Semantic keywords: "global diaspora network", "expat collaboration", "diaspora business network"

Mevcut paragraflari keyword-rich icerikle genislet (keyword stuffing yapmadan).

### Step 10: Footer FooterSection Semantic Fix + Trust Signals

**Dosya:** `src/components/FooterSection.tsx`

1. Footer h2'yi daha semantic yap (kalsin ama icerik zenginlestir)

3. Internal link cluster: Footer'a kategori linkleri ekle (sonra sayfalar eklenirse)

### Step 11: Preload + Performance Hints

**Dosya:** `index.html`

1. `<link rel="preload" href="/og-image.png" as="image">` (OG image icin)
2. Font preload: Inter font'u preload et
3. DNS prefetch: Supabase URL'i icin
