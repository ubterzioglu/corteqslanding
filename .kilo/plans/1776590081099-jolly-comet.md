# FAQ Section + SEO Content Fixes Plan

## Problem

SEO audit flags 3 errors on the landing page:
1. **Text depth** — main content needs ≥500 words (target: 700). Currently ~550 visible words.
2. **Headings & emphasis** — need at least one H2/H3 in question format ("Nedir / Nasıl…") and ≥2 semantic `<strong>`/`<em>` tags.
3. **First paragraph** — hero description is 1 sentence; needs 2–4 sentences as a proper summary.

The user wants a **visible accordion FAQ section** on the page using the 10 provided Q&As, which also resolves the word count issue.

---

## Current Page Structure

```
Index.tsx → HeroSection → AboutSection → CategoriesSection → AmbassadorSection → BloggerSection → SupportSection → FooterSection
```

**Available:** Radix UI `Accordion` component already installed at `src/components/ui/accordion.tsx`.

---

```tsx

</p>
```

This fixes: **"İlk paragrafı 2–4 cümleyle uzatın"**

The paragraph at line 20 already has one `<strong>`. Add a second emphasis with `<em>`:

```tsx

  topluluğunu güçlendiriyoruz.

```

This fixes: **"En az 2 yerde strong veya em kullanın"**

- Section H2 in question format: **"Sıkça Sorulan Sorular — CorteQS Nasıl Çalışır?"**
